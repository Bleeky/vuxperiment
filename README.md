# vuxperiment

A personnal project to experiment with VueJS and its ecosystem, as well as using the API of another project (serverlessxperiment), which is made using AWS Lambda.

Experimenting with:
- Vue Router
- Vuex
- Vuex ORM
- VeeValidate
- TailwindCSS
- CI using Gitlab
